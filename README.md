# Smart Farm Platform

Plateforme web intelligente de gestion agricole développée dans le cadre du PFA.

## Technologies utilisées

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Maven

### Frontend
- React.js

### Base de données
- PostgreSQL
